# AI Digital Product Black Book Deploy

This folder is ready to upload to Netlify, Vercel, GitHub Pages, Hostinger, or Cloudflare Pages.

## Files

- `index.html` - main web-book page
- `vercel.json` - Vercel static hosting config
- `assets/` - kept for hosting compatibility; the current book images are embedded inside `index.html`

## How To Deploy

Upload the full `ai-digital-product-black-book-deploy` folder to your hosting provider.

For most hosts, make sure `index.html` stays at the root of the deployed folder. After deployment, share the live `https://...` link on WhatsApp, Instagram, TikTok, YouTube, or anywhere else.

## Vercel Steps

1. Create a new GitHub repo.
2. Upload `index.html`, `vercel.json`, and `README.md` from this folder to the repo root.
3. Open Vercel and import that GitHub repo.
4. Keep framework as `Other`.
5. Leave build command and output directory empty.
6. Click Deploy.

## Local Test

Double-click `index.html` to open it in a browser.

## PDF Export

Open `index.html`, press `Ctrl + P`, choose `Save as PDF`, and enable background graphics for the best dark-theme PDF output.
